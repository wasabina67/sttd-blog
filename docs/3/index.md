## 紙の本に戻って感じた読書の楽しさ

最近は、電子書籍ではなく紙の本を手に取る時間が増えています。長いあいだ電子書籍を使っていましたが、改めて紙の本ならではの良さを感じるようになりました。

電子書籍はとても便利です。しかし、読み終わったあとに内容が少し頭に残りにくいと感じることがありました。その点、紙の本にはページをめくる感覚や、「このあたりに書いてあった」という空間的な記憶があり、内容を思い出しやすい気がします。

また、余白にメモを書き込んだり、付箋を貼ったりしながら読めるのも紙の本の魅力です。後から見返したときに、自分がどこに引っかかったのか、どこを大事だと思ったのかがすぐに分かるので、とても役立っています。

最近読んだのは、認知科学の観点から人間の記憶の仕組みを解説した本です。インプットとアウトプットのバランスや、効果的な復習のタイミングなどが実験結果をもとに説明されていて、非常に説得力がありました。

本を読む時間は、デジタルな情報から少し距離を置いて、一つのことに集中できる贅沢な時間だと感じています。これからもしばらくは、紙の本ならではの読書体験を楽しんでいきたいと思います。
